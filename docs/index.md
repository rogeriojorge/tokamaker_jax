# tokamaker-jax

`tokamaker-jax` is a staged JAX-native port of TokaMaker. It aims to preserve TokaMaker's Grad-Shafranov modeling capability while making the solver stack differentiable, easier to install, and friendlier for notebooks, CLI workflows, GUI exploration, and optimization loops.
