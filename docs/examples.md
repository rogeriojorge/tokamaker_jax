# Examples

The first executable example is a fixed-boundary seed solve:

```bash
tokamaker-jax examples/fixed_boundary.toml --plot outputs/fixed_boundary.png
```

For a wheel-only install without a repository checkout, export the packaged copy
first:

```bash
tokamaker-jax init-example fixed-boundary --output fixed_boundary.toml
tokamaker-jax fixed_boundary.toml --plot outputs/fixed_boundary.png
```

Benchmark and literature-reproduction examples write JSON reports and figures
that can be archived with a run:

```bash
python examples/benchmark_report.py \
  --output outputs/benchmark_report.json \
  --thresholds docs/validation/benchmark_thresholds.json \
  --comparison-output outputs/benchmark_threshold_report.json
python examples/benchmark_history.py \
  outputs/benchmark_report.json \
  outputs/benchmark_history.jsonl \
  --threshold-report outputs/benchmark_threshold_report.json
python examples/reproduce_cpc_seed_family.py outputs/literature/cpc_seed_family
```
