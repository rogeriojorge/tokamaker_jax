# Examples

The first executable example is a fixed-boundary seed solve:

```bash
tokamaker-jax examples/fixed_boundary.toml --plot outputs/fixed_boundary.png
```

```{image} _static/fixed_boundary_seed.png
:alt: Fixed-boundary seed equilibrium contours
```

```{image} _static/pressure_sweep.gif
:alt: Pressure sweep animation
```

The first geometry preview shows the TOML/Python region primitives that will
feed TokaMaker-style machine definitions and mesh generation:

```{image} _static/region_geometry_seed.png
:alt: Region geometry preview with plasma, vacuum vessel, and PF coil
```

The first closed-form circular-loop free-boundary kernel is checked against a
high-resolution toroidal quadrature reference:

```bash
tokamaker-jax verify --gate circular-loop
```

```{image} _static/circular_loop_elliptic_response.png
:alt: Closed-form circular-loop elliptic response
```

The first nonlinear triangular-FEM profile iteration example is generated by
the documentation asset script and is also exercised by the
`profile-iteration` validation gate:

```bash
tokamaker-jax verify --gate profile-iteration
```

```{image} _static/profile_iteration.png
:alt: Nonlinear p=1 FEM profile iteration contours
```

Benchmark and literature-reproduction examples write JSON reports and figures
that can be archived with a run:

```bash
python examples/benchmark_report.py --output outputs/benchmark_report.json
python examples/reproduce_cpc_seed_family.py outputs/literature/cpc_seed_family
```

```{image} _static/cpc_seed_family.png
:alt: CPC seed-family reproduction surrogate
```
